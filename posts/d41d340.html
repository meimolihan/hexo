⚙️ 共享配置

通过 Web 界面配置：

  1. 登录 iStoreOS 管理界面
  2. 进入”服务” → “网络共享”
  3. 点击”添加”按钮创建新共享
  4. 配置以下参数：
    • 名称: 共享显示名称（如：mydisk）
    • 路径: /mnt/mydisk
    • 允许用户: admin（或选择用户组）
    • 权限: 读写/只读
    • 允许匿名访问: 根据需求选择
    • 可浏览: 是/否
  5. 点击”保存并应用”

🚀 客户端访问指南

Windows 客户端

  1. 打开”文件资源管理器”
  2. 在地址栏输入: \\<iStoreOS-IP>\mydisk
  3. 输入用户名和密码: admin / [你的密码]
  4. 勾选”记住我的凭据”
  5. 如需映射网络驱动器：
    • 右键点击”此电脑” → “映射网络驱动器”
    • 选择驱动器字母
    • 输入文件夹路径: \\<iStoreOS-IP>\mydisk
    • 勾选”使用其他凭据连接”

macOS 客户端

  1. 打开”访达”
  2. 点击”前往” → “连接服务器”
  3. 输入: smb://<iStoreOS-IP>/mydisk
  4. 输入用户名和密码
  5. 如需自动挂载：
    • 系统偏好设置 → 用户与群组 → 登录项
    • 添加 Samba 共享

Linux 客户端

移动设备访问

  • iOS:

    1. 使用”文件”应用
    2. 点击右上角”···” → “连接服务器”
    3. 输入: smb://<iStoreOS-IP>
    4. 输入用户名和密码

  • Android:

    1. 安装”Solid Explorer”或”CX File Explorer”
    2. 添加新存储 → LAN/Windows Network
    3. 输入服务器地址和凭据
