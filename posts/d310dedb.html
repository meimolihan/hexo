  • рҹҢҗ MinDoc е®ҳзҪ‘ең°еқҖпјҡhttps://www.gopeed.com/zh-CN

Gopeed жҳҜдёҖж¬ҫејҖжәҗзҡ„й«ҳжҖ§иғҪдёӢиҪҪз®ЎзҗҶеҷЁпјҢж”ҜжҢҒ HTTPгҖҒHTTPSгҖҒFTPгҖҒBTгҖҒзЈҒеҠӣй“ҫжҺҘзӯүеӨҡз§ҚдёӢиҪҪеҚҸи®®гҖӮйҖҡиҝҮ Docker йғЁзҪІпјҢжӮЁеҸҜд»ҘиҪ»жқҫжҗӯе»әдёҖдёӘеҠҹиғҪејәеӨ§зҡ„дёӢиҪҪжңҚеҠЎеҷЁпјҢж”ҜжҢҒиҝңзЁӢз®ЎзҗҶе’ҢеӨҡи®ҫеӨҮи®ҝй—®гҖӮ

рҹӣ пёҸ ж ёеҝғзү№жҖ§

зү№жҖ§зұ»еҲ« еҠҹиғҪжҸҸиҝ° дјҳеҠҝ
рҹҢҗ еӨҡеҚҸи®®ж”ҜжҢҒ HTTPгҖҒHTTPSгҖҒFTPгҖҒBTгҖҒзЈҒеҠӣй“ҫжҺҘе…Ёйқўж”ҜжҢҒ дёҖз«ҷејҸдёӢиҪҪи§ЈеҶіж–№жЎҲ
вҡЎ й«ҳжҖ§иғҪдёӢиҪҪ еӨҡзәҝзЁӢдёӢиҪҪпјҢе……еҲҶеҲ©з”ЁеёҰе®Ҫ жңҖеӨ§еҢ–дёӢиҪҪйҖҹеәҰ
рҹ“ұ Web з•Ңйқў зӣҙи§Ӯзҡ„жөҸи§ҲеҷЁз®ЎзҗҶз•Ңйқў ж— йңҖе®үиЈ…е®ўжҲ·з«ҜпјҢи·Ёе№іеҸ°и®ҝй—®
рҹ”„ ж–ӯзӮ№з»ӯдј  ж”ҜжҢҒдёӢиҪҪдёӯж–ӯеҗҺз»§з»ӯдёӢиҪҪ иҠӮзңҒж—¶й—ҙе’ҢжөҒйҮҸ
рҹҺҜ д»»еҠЎз®ЎзҗҶ жү№йҮҸдёӢиҪҪгҖҒд»»еҠЎжҺ’йҳҹгҖҒйҖҹеәҰйҷҗеҲ¶ зІҫз»ҶеҢ–дёӢиҪҪжҺ§еҲ¶
рҹҗі е®№еҷЁеҢ–йғЁзҪІ дҪҝз”Ё Docker з®ҖеҢ–е®үиЈ…е’Ңз»ҙжҠӨ зҺҜеўғйҡ”зҰ»пјҢйғЁзҪІз®ҖеҚ•
рҹ”’ е®үе…Ёи®ҝй—® еҸҜйҖүзҡ„иә«д»ҪйӘҢиҜҒдҝқжҠӨ дҝқйҡңдёӢиҪҪжңҚеҠЎе®үе…Ё

1. еҲӣе»әйғЁзҪІзӣ®еҪ•е’Ң docker-compose.yml ж–Үд»¶ рҹ“Ғ

вҡҷпёҸ еҸҜйҖүзҺҜеўғеҸҳйҮҸпјҡ

  • GOPEED_WEB_AUTH=false # жҳҜеҗҰеҗҜз”Ёи®ӨиҜҒ
  • GOPEED_WEB_USERNAME=admin # з”ЁжҲ·еҗҚ
  • GOPEED_WEB_PASSWORD=password # еҜҶз Ғ

2. жӢүеҸ–й•ңеғҸе№¶еҗҜеҠЁе®№еҷЁ рҹҡҖ

рҹҡҖ дәҢгҖҒдҪҝз”Ё Gopeed

1. и®ҝй—® Web з•Ңйқў рҹ’»

  • жң¬ең°и®ҝй—®: http://дҪ зҡ„жңҚеҠЎеҷЁIP:6600
  • еҹҹеҗҚи®ҝй—®: https://download.yourdomain.com (й…ҚзҪ®еҸҚеҗ‘д»ЈзҗҶеҗҺ)

2. йҰ–ж¬ЎдҪҝз”ЁжӯҘйӘӨ рҹҺҜ

  1. жү“ејҖжөҸи§ҲеҷЁи®ҝй—®дёҠиҝ°ең°еқҖ
  2. ж №жҚ®з•ҢйқўжҸҗзӨәиҝӣиЎҢеҲқе§Ӣи®ҫзҪ®пјҲеҰӮжһңжңүпјү
  3. й…ҚзҪ®дёӢиҪҪзӣ®еҪ•е’Ңй»ҳи®Өи®ҫзҪ®
  4. ејҖе§Ӣж·»еҠ дёӢиҪҪд»»еҠЎ

йҖҡиҝҮжң¬ж–ҮжЎЈпјҢдҪ е·Із»ҸжҲҗеҠҹдҪҝз”Ё Docker йғЁзҪІдәҶ Gopeed дёӢиҪҪеҷЁпјҢе№¶йҖҡиҝҮ Nginx еҸҚеҗ‘д»ЈзҗҶе®һзҺ°дәҶе®үе…ЁдҫҝжҚ·зҡ„иҝңзЁӢи®ҝй—®гҖӮзҺ°еңЁпјҢдҪ еҸҜд»Ҙпјҡ

вҡЎ жһҒйҖҹдёӢиҪҪ дә«еҸ—еӨҡеҚҸи®®еӨҡзәҝзЁӢеёҰжқҘзҡ„жһҒиҮҙдёӢиҪҪдҪ“йӘҢпјҢжңҖеӨ§еҢ–еҲ©з”ЁзҪ‘з»ңеёҰе®Ҫ
рҹҢҗ иҝңзЁӢз®ЎзҗҶ йҡҸж—¶йҡҸең°йҖҡиҝҮ Web з•Ңйқўж·»еҠ е’Ңз®ЎзҗҶдёӢиҪҪд»»еҠЎпјҢжү“з ҙи®ҫеӨҮйҷҗеҲ¶
рҹ”„ зЁіе®ҡеҸҜйқ  ж”ҜжҢҒж–ӯзӮ№з»ӯдј е’Ңд»»еҠЎйҳҹеҲ—пјҢзЎ®дҝқйҮҚиҰҒдёӢиҪҪдёҚдјҡдёӯж–ӯ
рҹ”’ е®үе…ЁеҠ еҜҶ йҖҡиҝҮ HTTPS еҠ еҜҶиҝһжҺҘдҝқжҠӨдҪ зҡ„дёӢиҪҪйҡҗз§Ғе’Ңдј иҫ“е®үе…Ё

дёӢиҪҪдёҚд»…жҳҜиҺ·еҸ–иө„жәҗзҡ„ж–№ејҸпјҢжӣҙжҳҜж•°еӯ—з”ҹжҙ»зҡ„й«ҳж•ҲеҠ©жүӢгҖӮж„ҝ Gopeed жҲҗдёәдҪ еҸҜйқ зҡ„дёӢиҪҪдјҷдјҙпјҢи®©дҪ зҡ„жҜҸдёҖж¬Ўиө„жәҗиҺ·еҸ–йғҪеҸҳеҫ—з®ҖеҚ•гҖҒеҝ«йҖҹгҖҒзЁіе®ҡгҖӮ
