3гҖҒеўһеҠ  Git зҡ„и¶…ж—¶ж—¶й—ҙ

4гҖҒжҹҘзңӢеҪ“еүҚй…ҚзҪ®

дёү гҖҒеҲҮжҚўеҲ°SSHеҜҶй’ҘпјҲжҺЁиҚҗпјү

2гҖҒ еҰӮжһңжІЎжңүSSHеҜҶй’ҘпјҢеҲӣе»әж–°зҡ„SSHеҜҶй’Ҙ

иҫ“вҖ®д»Ҙе…ҘвҖ¬дёӢе‘Ҫд»Өз”ҹжҲҗSSHвҖ®й’ҘеҜҶвҖ¬еҜ№пјҡ

3гҖҒдёҖй”®еӨҚеҲ¶е…¬й’Ҙж–Үд»¶еҲ°еүӘеҲҮжқҝ

4гҖҒ е°ҶвҖ®й’Ҙе…¬вҖ¬ж·»еҠ еҲ°GitHubиҙҰжҲ·

  • зҷ»еҪ•вҖ®дҪ еҲ°вҖ¬зҡ„GitHubиҙҰжҲ·гҖӮ
  • зӮ№еҮ»еҸідёҠи§’зҡ„еӨҙеғҸпјҢйҖүжӢ© SettingsгҖӮ
  • еңЁвҖ®дҫ§е·ҰвҖ¬иҸңеҚ•дёӯйҖүжӢ© SSH and GPG keysгҖӮ
  • зӮ№еҮ» New SSH key жҢүй’®гҖӮ
  • еңЁ Title еӯ—ж®өдёӯиҫ“вҖ®дёҖе…ҘвҖ¬дёӘеҗҚз§°пјҲдҫӢеҰӮпјҢдҪ зҡ„з”өи„‘еҗҚз§°пјүпјҢ
  • еңЁ вҖңKeyвҖқ еӯ—ж®өдёӯзІҳиҙҙеҲҡжүҚвҖ®еҲ¶еӨҚвҖ¬зҡ„е…¬й’ҘеҶ…е®№пјҢ
  • з„¶еҗҺзӮ№еҮ» вҖңAdd SSH keyвҖқ жҢүй’®гҖӮ



5гҖҒдҝ®ж”№иҝңзЁӢд»“еә“URLдёәSSHж јејҸ

3гҖҒе°Ҷжң¬ең°д»“еә“дёҺ GitHub дёҠзҡ„иҝңзЁӢд»“еә“е…іиҒ”е№¶жҺЁйҖҒд»Јз Ғ

4гҖҒе°Ҷжң¬ең°д»“еә“жҺЁйҖҒеҲ° GitHub (жӣҙж–°д»“еә“)

branch 'main' set up to track 'origin/main'.
иҜҙжҳҺжң¬ең°зҡ„ main еҲҶж”Ҝе·Із»Ҹи®ҫзҪ®дёәи·ҹиёӘиҝңзЁӢд»“еә“ origin зҡ„ main еҲҶж”ҜгҖӮеҗҺз»ӯеҶҚдҪҝз”Ё git push жҲ– git pull ж—¶пјҢеҰӮжһңдёҚжҢҮе®ҡеҲҶж”ҜпјҢGit е°ұдјҡй»ҳи®Өж“ҚдҪңиҝҷдёӘи·ҹиёӘе…ізі»еҜ№еә”зҡ„еҲҶж”ҜгҖӮ

5гҖҒжҹҘзңӢзҠ¶жҖҒ

жҹҘзңӢеҪ“еүҚе·ҘдҪңеҢәе’ҢжҡӮеӯҳеҢәзҡ„зҠ¶жҖҒпјҢдәҶи§Је“Әдәӣж–Үд»¶е·Ідҝ®ж”№гҖҒж–°еўһжҲ–еҲ йҷӨгҖӮ

дә” гҖҒGit е…ӢйҡҶйЎ№зӣ®

1гҖҒGit httpе…ӢйҡҶйЎ№зӣ®

2гҖҒGit SSHе…ӢйҡҶйЎ№зӣ®

  • жҸҗзӨәпјҡAre you sure you want to continue connecting (yes/no/[fingerprint])? yes

  • еҲӣе»әжҲ–зј–иҫ‘ config ж–Үд»¶пјҢCMDе‘Ҫд»Өпјҡ

  1. йў„е…Ҳж·»еҠ  GitHub дё»жңәдҝЎжҒҜ
  • иӢҘи§ЈеҶі .ssh зӣ®еҪ•й—®йўҳеҗҺд»ҚдёҚжғіжҜҸж¬ЎжүӢеҠЁзЎ®и®ӨпјҢеҸҜйў„е…Ҳе°Ҷ GitHub зҡ„дё»жңәдҝЎжҒҜж·»еҠ еҲ° known_hosts ж–Үд»¶дёӯгҖӮ
  • иҺ·еҸ– GitHub зҡ„ SSH жҢҮзә№пјҡ

иҜҘе‘Ҫд»Өдјҡиҫ“еҮә GitHub зҡ„ ED25519 еҜҶй’ҘжҢҮзә№дҝЎжҒҜгҖӮ

  • е°ҶжҢҮзә№дҝЎжҒҜж·»еҠ еҲ° known_hosts ж–Үд»¶пјҡ

жӯӨе‘Ҫд»ӨдјҡжҠҠ GitHub зҡ„ SSH жҢҮзә№дҝЎжҒҜиҝҪеҠ еҲ° known_hosts ж–Үд»¶дёӯгҖӮд№ӢеҗҺеҶҚиҝӣиЎҢе…ӢйҡҶж“ҚдҪңж—¶пјҢе°ұдёҚдјҡеҶҚжҸҗзӨәдҪ зЎ®и®Өдё»жңәдҝЎжҒҜдәҶгҖӮ

е…ӯ гҖҒеҝҪз•Ҙж–Үд»¶

1гҖҒеҲӣе»ә .gitignore ж–Үд»¶

2гҖҒжҢҮе®ҡдёҚйңҖиҰҒиў« Git и·ҹиёӘзҡ„ж–Үд»¶жҲ–зӣ®еҪ•гҖӮдҫӢеҰӮпјҡ

дёғ гҖҒGit д»“еә“еҗҲе№¶

1 гҖҒе°ҶвҖ®вҖ¬гҖҗжң¬ең°д»“еә“гҖ‘зҡ„жӣҙж”№еҗҲвҖ®еҲ°е№¶вҖ¬гҖҗзЁӢиҝңд»“еә“гҖ‘

  1. жҹҘзңӢжҳҜеҗҰжңүжңӘжҸҗдәӨзҡ„дҝ®ж”№

  2. жӣҙж–°жң¬ең°еҲҶж”ҜеҲ°жңҖж–°зҠ¶жҖҒ

  3. жҸҗдәӨжң¬ең°жӣҙж”№

  4. жҺЁйҖҒжӣҙж”№еҲ°иҝңзЁӢд»“еә“

2 гҖҒе°ҶгҖҗвҖ®зЁӢиҝңвҖ¬д»“еә“гҖ‘зҡ„жӣҙж”№еҗҲвҖ®еҲ°е№¶вҖ¬гҖҗжң¬ең°д»“еә“гҖ‘

  1. жҹҘзңӢеҪ“еүҚеҲҶж”Ҝ
    йҰ–е…ҲпјҢзЎ®и®ӨдҪ еҪ“вҖ®жүҖеүҚвҖ¬еңЁзҡ„еҲҶж”ҜгҖӮеҰӮжһңдҪ жү“вҖ®е°Ҷз®—вҖ¬иҝңзЁӢвҖ®ж”№жӣҙвҖ¬еҗҲе№¶еҲ° main еҲҶж”ҜпјҢзЎ®дҝқдҪ еңЁиҜҘеҲҶж”ҜдёҠгҖӮ

  2. еҲҮжҚўеҲ°зӣ®ж ҮеҲҶж”Ҝ

  3. зӣҙжҺҘиҺ·вҖ®е№¶еҸ–вҖ¬еҗҲе№¶иҝңвҖ®жӣҙзЁӢвҖ¬ж”№

  1. жҹҘвҖ®жңҖзңӢвҖ¬ж–°жҸҗдәӨи®°еҪ•
    жҹҘзңӢвҖ®ж–°жңҖвҖ¬зҡ„жҸҗдәӨи®°еҪ•пјҢзЎ®вҖ®иҝңи®ӨвҖ¬зЁӢд»“еә“зҡ„жӣҙвҖ®жҳҜж”№вҖ¬еҗҰе·Із»ҸжӯЈзЎ®еҗҲвҖ®еҲ°е№¶вҖ¬жң¬ең°гҖӮ

е…« гҖҒжё…зҗҶд»“еә“е№¶дјҳеҢ–еӯҳеӮЁ

1гҖҒжЈҖжҹҘд»“еә“еӨ§е°Ҹ

дҪ еҸҜд»ҘдҪҝз”Ёд»ҘдёӢе‘Ҫд»ӨжЈҖжҹҘд»“еә“зҡ„еӨ§е°ҸеҸҳеҢ–пјҡ

2гҖҒжё…зҗҶд»“еә“е№¶дјҳеҢ–еӯҳеӮЁ

git gc зҡ„дё»иҰҒеҠҹиғҪжҳҜжё…зҗҶжң¬ең°д»“еә“дёӯзҡ„ж— з”ЁеҜ№иұЎпјҲеҰӮжңӘеј•з”Ёзҡ„жҸҗдәӨгҖҒж ‘еҜ№иұЎгҖҒblob еҜ№иұЎзӯүпјүпјҢе№¶дјҳеҢ–еӯҳеӮЁз»“жһ„пјҲеҰӮеҺӢзј©еҜ№иұЎгҖҒдјҳеҢ– packfile зӯүпјүгҖӮ
иҝҷдәӣж“ҚдҪңдёҚдјҡж”№еҸҳдҪ зҡ„д»Јз ҒеҶ…е®№гҖҒеҲҶж”Ҝз»“жһ„жҲ–жҸҗдәӨеҺҶеҸІгҖӮеӣ жӯӨпјҢиҝңзЁӢд»“еә“зҡ„еҶ…е®№дёҚдјҡеӣ дёәжң¬ең°жү§иЎҢ git gc иҖҢеҸ‘з”ҹеҸҳеҢ–гҖӮ

3гҖҒиҮӘеҠЁз»ҙжҠӨ

4гҖҒжЈҖжҹҘ Git д»“еә“зҡ„е®Ңж•ҙжҖ§

е»әи®®е®ҡжңҹиҝҗиЎҢ git gc е’Ң git fsckпјҢд»ҘдҝқжҢҒд»“еә“зҡ„ж•ҙжҙҒе’ҢеҒҘеә·гҖӮеҸҜд»Ҙи®ҫзҪ® gc.auto еҸӮж•°жқҘиҮӘеҠЁи§ҰеҸ‘еһғеңҫеӣһж”¶пјҡ
иҝҷиЎЁзӨәеҪ“д»“еә“дёӯзҡ„жқҫж•ЈеҜ№иұЎж•°йҮҸиҫҫеҲ° 250 ж—¶пјҢGit дјҡиҮӘеҠЁиҝҗиЎҢ git gcгҖӮ

д№қ гҖҒgit еҲҶж”Ҝж“ҚдҪң

1гҖҒ жҹҘзңӢеҲҶж”Ҝ

  • жҹҘзңӢжүҖжңүеҲҶж”ҜпјҲеҢ…жӢ¬жң¬ең°е’ҢиҝңзЁӢеҲҶж”Ҝпјүпјҡ

  • жҹҘзңӢжң¬ең°еҲҶж”Ҝgit branch пјҢжҹҘзңӢиҝңзЁӢеҲҶж”Ҝgit branch -r

2гҖҒ еҲӣе»әеҲҶж”Ҝ

  • еҲӣе»әдёҖдёӘж–°зҡ„еҲҶж”ҜпјҲдҪҶдёҚеҲҮжҚўеҲ°иҜҘеҲҶж”Ҝпјүпјҡ

  • еҲҮжҚўеҲҶж”Ҝ

  • еҲӣе»әе№¶еҲҮжҚўеҲ°ж–°еҲҶж”Ҝпјҡ

3гҖҒ еҲ йҷӨеҲҶж”Ҝ

  • еҲ йҷӨжң¬ең°еҲҶж”Ҝпјҡ

  • пјҲеҰӮжһңеҲҶж”ҜжңӘеҗҲе№¶пјҢдјҡжҸҗзӨәй”ҷиҜҜпјӣејәеҲ¶еҲ йҷӨз”Ё -Dпјү

  • еҲ йҷӨиҝңзЁӢеҲҶж”Ҝпјҡ

4гҖҒ йҮҚе‘ҪеҗҚеҲҶж”Ҝ

  • йҮҚе‘ҪеҗҚеҪ“еүҚеҲҶж”Ҝпјҡ

  • йҮҚе‘ҪеҗҚе…¶д»–еҲҶж”Ҝпјҡ

5гҖҒ жҺЁйҖҒеҲҶж”Ҝ

  • жҺЁйҖҒжң¬ең°еҲҶж”ҜеҲ°иҝңзЁӢд»“еә“пјҡ

  • и®ҫзҪ®дёҠжёёеҲҶж”ҜпјҲе…іиҒ”жң¬ең°еҲҶж”ҜдёҺиҝңзЁӢеҲҶж”Ҝпјүпјҡ

6гҖҒ жӢүеҸ–еҲҶж”Ҝ

7гҖҒ и·ҹиёӘиҝңзЁӢеҲҶж”Ҝ

  • жҹҘзңӢжң¬ең°еҲҶж”ҜжҳҜеҗҰи·ҹиёӘиҝңзЁӢеҲҶж”Ҝпјҡ

  • и®ҫзҪ®жң¬ең°еҲҶж”Ҝи·ҹиёӘиҝңзЁӢеҲҶж”Ҝпјҡ

8гҖҒ и§ЈеҶіеҶІзӘҒ

  • еңЁеҗҲе№¶жҲ– rebase ж—¶еҸҜиғҪдјҡеҮәзҺ°еҶІзӘҒпјҢи§ЈеҶіеҶІзӘҒеҗҺйңҖиҰҒжүӢеҠЁж Үи®°ж–Үд»¶дёәе·Іи§ЈеҶіпјҡ

еҚҒ гҖҒGit дҪҝз”Ё ssh еҜҶй’ҘиҝһжҺҘ GitHub

1гҖҒ з”ҹжҲҗдёӘдәәи®ҝй—®д»ӨзүҢ

  • зҷ»еҪ•еҲ°дҪ зҡ„GitHubиҙҰжҲ·гҖӮ
  • зӮ№еҮ»вҖ®дёҠеҸівҖ¬и§’зҡ„еӨҙеғҸпјҢйҖүжӢ© SettingsгҖӮ
  • еңЁе·Ұдҫ§иҸңвҖ®дёӯеҚ•вҖ¬йҖүжӢ© Developer settingsгҖӮ
  • еңЁе·ҰвҖ®иҸңдҫ§вҖ¬еҚ•дёӯйҖүжӢ© Personal access tokensгҖӮ
  • зӮ№еҮ» Generate new token жҢүй’®гҖӮ
  • з»ҷд»ӨзүҢе‘ҪеҗҚпјҲдҫӢеҰӮ Git-Push-TokenпјүпјҢз„¶еҗҺж №жҚ®дҪ вҖ®йңҖзҡ„вҖ¬жұӮйҖүжӢ©жқғйҷҗиҢғеӣҙгҖӮеҜ№дәҺз®ҖеҚ•вҖ®жҺЁзҡ„вҖ¬йҖҒж“ҚдҪңпјҢйҖҡеёёеҸӘйңҖиҰҒ repo жқғйҷҗеҚіеҸҜгҖӮ
  • зӮ№еҮ» Generate token жҢүй’®пјҢз„¶еҗҺвҖ®еҲ¶еӨҚвҖ¬з”ҹжҲҗзҡ„д»ӨзүҢгҖӮ
