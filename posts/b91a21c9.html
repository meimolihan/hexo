📖 简介

1Panel 是一款现代化的开源 Linux 服务器运维管理面板，通过 Web 界面轻松管理服务器、网站、数据库、容器等。它提供了丰富的功能和直观的操作界面，是服务器管理的得力助手。

网盘分享： https://pan.quark.cn/s/56eb82ad45cc

✨ 主要特性

  • 🌐 Web 管理界面：直观的图形化操作界面
  • 🐳 容器管理：支持 Docker 和容器管理
  • 📦 应用商店：一键安装常用应用，支持选择远程数据库
  • 🛠️ Node.js 支持：v1.7.0 版本新增 Node.js 运行环境
  • 🗑️ 缓存清理：支持系统缓存清理功能，释放磁盘空间
  • 🔒 安全防护：内置防火墙和安全设置
  • 📊 监控统计：实时服务器状态监控
  • 💾 备份恢复：完整的备份和恢复功能
  • 🚀 高性能：基于 Golang 开发，响应迅速
