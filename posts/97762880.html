Proxmox VE жҢӮиҪҪе·Іжңүж•°жҚ®зҡ„ ext4 е’Ң NTFS зЎ¬зӣҳ рҹҗ§

жң¬жҢҮеҚ—иҜҰз»Ҷи®Іи§ЈеҰӮдҪ•еңЁ Proxmox VE дёӯж— жҚҹжҢӮиҪҪе·ІеҢ…еҗ«ж•°жҚ®зҡ„ ext4 е’Ң NTFS ж јејҸзЎ¬зӣҳпјҢж— йңҖж јејҸеҢ–еҚіеҸҜзӣҙжҺҘи®ҝй—®е’ҢдҪҝз”ЁзҺ°жңүж•°жҚ®гҖӮе®ҢзҫҺи§ЈеҶі PVE Web йқўжқҝж— жі•зӣҙжҺҘжҢӮиҪҪе·Іжңүж•°жҚ®зЎ¬зӣҳзҡ„й—®йўҳпјҒ

рҹ“Ӣ еҜјиҲӘзӣ®еҪ•

  • вңЁ жҰӮиҝ°
  • вңЁ еҶ…е®№зү№зӮ№
  • рҹҗ§ дёҖжҢӮиҪҪ-ext4-ж јејҸзЎ¬зӣҳ
  • рҹ’ҫ дәҢжҢӮиҪҪ-ntfs-ж јејҸзЎ¬зӣҳ
  • вҡҷпёҸ дёүй«ҳзә§й…ҚзҪ®дёҺдјҳеҢ–
  • рҹ”§ еӣӣеёёи§Ғй—®йўҳдёҺи§ЈеҶіж–№жЎҲ
  • рҹ“Ҡ дә”ж–Үд»¶зі»з»ҹзү№жҖ§еҜ№жҜ”
  • рҹ’Ў е…ӯжңҖдҪіе®һи·өе»әи®®
  • рҹҡЁ дёғжіЁж„ҸдәӢйЎ№дёҺиӯҰе‘Ҡ

вңЁ жҰӮиҝ°

Proxmox VE Web йқўжқҝй»ҳи®ӨеҸӘиғҪж·»еҠ ж–°зЎ¬зӣҳе№¶еҲқе§ӢеҢ–пјҢж— жі•зӣҙжҺҘжҢӮиҪҪе·Іж јејҸеҢ–зҡ„еҗ«жңүж•°жҚ®зҡ„зЎ¬зӣҳгҖӮиҝҷз»ҷйңҖиҰҒиҝҒз§»зҺ°жңүж•°жҚ®зҡ„з”ЁжҲ·еёҰжқҘдәҶдёҚдҫҝгҖӮжң¬жҢҮеҚ—иҜҰз»Ҷд»Ӣз»ҚеҰӮдҪ•йҖҡиҝҮе‘Ҫд»ӨиЎҢжүӢеҠЁжҢӮиҪҪ ext4 е’Ң NTFS ж јејҸзҡ„зЎ¬зӣҳпјҢе№¶дҪҝе…¶еңЁ PVE дёӯеҸҜз”ЁпјҢеҗҢж—¶дҝқжҢҒеҺҹжңүж•°жҚ®е®Ңж•ҙж— жҚҹгҖӮ

йҖӮз”ЁеңәжҷҜпјҡ

  • рҹ”„ д»Һе…¶д»–зі»з»ҹиҝҒз§»ж•°жҚ®еҲ° PVE
  • рҹ’ҫ зӣҙжҺҘи®ҝй—® Windows зі»з»ҹдёӯзҡ„ж–Үд»¶
  • рҹ—ӮпёҸ е…ұдә«еӯҳеӮЁз©әй—ҙ between multiple containers
  • рҹ“Җ еҲ©з”ЁзҺ°жңүзЎ¬зӣҳжү©еұ• PVE еӯҳеӮЁ

вңЁ еҶ…е®№зү№зӮ№

  • рҹҡҖ ж— жҚҹжҢӮиҪҪпјҡж— йңҖж јејҸеҢ–пјҢдҝқз•ҷеҺҹжңүж•°жҚ®е®Ңж•ҙ
  • рҹҺҜ еҸҢзі»з»ҹж”ҜжҢҒпјҡеҗҢж—¶иҰҶзӣ– ext4 (Linux) е’Ң NTFS (Windows) ж–Үд»¶зі»з»ҹ
  • вҡЎ жҖ§иғҪдјҳеҢ–пјҡжҸҗдҫӣй’ҲеҜ№жңәжў°зЎ¬зӣҳе’Ң SSD зҡ„дё“з”ЁдјҳеҢ–ж–№жЎҲ
  • рҹ”җ жқғйҷҗз®ЎзҗҶпјҡиҜҰз»Ҷи®Іи§ЈеӨҡз”ЁжҲ·зҺҜеўғдёӢзҡ„жқғйҷҗй…ҚзҪ®
  • рҹ“Ӯ дёӯж–Үж”ҜжҢҒпјҡи§ЈеҶідёӯж–Үж–Үд»¶еҗҚд№ұз Ғй—®йўҳ

  • рҹ”„ иҮӘеҠЁжҢӮиҪҪпјҡй…ҚзҪ®ејҖжңәиҮӘеҠЁжҢӮиҪҪпјҢдёҖеҠіж°ёйҖё

  • рҹ“ҡ еҸӮиҖғиө„жәҗ

    • рҹ“– еӣҫж–Үж•ҷзЁӢ
    • рҹҺҘ и§Ҷйў‘ж•ҷзЁӢ

рҹҗ§ дёҖгҖҒжҢӮиҪҪ ext4 ж јејҸзЎ¬зӣҳ

1. жҹҘзңӢжүҖжңүзЈҒзӣҳи®ҫеӨҮ

жӯӨе‘Ҫд»Өе°ҶеҲ—еҮәжүҖжңүиҝһжҺҘзҡ„зЈҒзӣҳи®ҫеӨҮеҸҠе…¶еҲҶеҢәжғ…еҶөпјҢеё®еҠ©жӮЁиҜҶеҲ«иҰҒжҢӮиҪҪзҡ„зЈҒзӣҳгҖӮ

2. жҹҘзңӢе…·дҪ“ж–Үд»¶зі»з»ҹдҝЎжҒҜ

е°Ҷ /dev/sda жӣҝжҚўдёәжӮЁзҡ„е®һйҷ…и®ҫеӨҮеҗҚз§°пјҢжҹҘзңӢж–Үд»¶зі»з»ҹзұ»еһӢе’Ң UUIDгҖӮ

3. еҲӣе»әжҢӮиҪҪзӮ№е№¶жҢӮиҪҪзЎ¬зӣҳ

е°Ҷ /dev/sda1 жӣҝжҚўдёәжӮЁзҡ„е®һйҷ…еҲҶеҢәпјҢ/mnt/mydisk жӣҝжҚўдёәжӮЁе–ңж¬ўзҡ„жҢӮиҪҪи·Ҝеҫ„гҖӮ

4. еҸ–ж¶ҲжҢӮиҪҪпјҲеҰӮйңҖпјү

е®үе…ЁеҚёиҪҪеҲҶеҢәпјҢзЎ®дҝқж•°жҚ®е®Ңж•ҙжҖ§гҖӮ

5. й…ҚзҪ®ејҖжңәиҮӘеҠЁжҢӮиҪҪ

5.1 жҹҘзңӢзЎ¬зӣҳзҡ„ UUID

и®°еҪ•дёӢжӮЁиҰҒжҢӮиҪҪзҡ„еҲҶеҢәзҡ„ UUIDгҖӮ

5.2 зј–иҫ‘ fstab ж–Үд»¶й…ҚзҪ®иҮӘеҠЁжҢӮиҪҪ

ж·»еҠ д»ҘдёӢиЎҢпјҲж №жҚ®е®һйҷ…жғ…еҶөдҝ®ж”№ UUID е’ҢжҢӮиҪҪзӮ№пјүпјҡ

дҪҝз”Ё nofail йҖүйЎ№йҳІжӯўеӣ жүҫдёҚеҲ°зЎ¬зӣҳиҖҢеҜјиҮҙзі»з»ҹеҗҜеҠЁеӨұиҙҘгҖӮ

рҹ’ҫ дәҢгҖҒжҢӮиҪҪ NTFS ж јејҸзЎ¬зӣҳпјҲWindows зЎ¬зӣҳпјү

1. жҹҘзңӢзЈҒзӣҳдҝЎжҒҜ

2. жҹҘзңӢ NTFS еҲҶеҢәиҜҰз»ҶдҝЎжҒҜ

3. е®үиЈ… NTFS ж”ҜжҢҒиҪҜд»¶еҢ…

NTFS-3G жҳҜејҖжәҗзҡ„ NTFS й©ұеҠЁзЁӢеәҸпјҢе…Ғи®ё Linux зі»з»ҹиҜ»еҶҷ NTFS ж јејҸзЎ¬зӣҳгҖӮ

4. еҲӣе»әжҢӮиҪҪзӮ№е№¶дҝ®еӨҚ NTFS ж–Үд»¶зі»з»ҹ

ntfsfix з”ЁдәҺдҝ®еӨҚеёёи§Ғзҡ„ NTFS й—®йўҳпјҢдҪҶдёҚдјҡжҚҹеқҸзҺ°жңүж•°жҚ®гҖӮ

5. жҢӮиҪҪ NTFS еҲҶеҢә

дҪҝз”Ёй»ҳи®ӨеҸӮж•°жҢӮиҪҪ NTFS еҲҶеҢәгҖӮ

6. йӘҢиҜҒжҢӮиҪҪз»“жһң

7. еңЁ PVE Web з•Ңйқўдёӯж·»еҠ еӯҳеӮЁ

  1. вң… зҷ»еҪ• PVE Web з®ЎзҗҶз•Ңйқў
  2. вң… иҝӣе…ҘвҖқж•°жҚ®дёӯеҝғвҖқ вҶ’ вҖңеӯҳеӮЁвҖқ вҶ’ вҖңж·»еҠ вҖқ вҶ’ вҖңзӣ®еҪ•вҖқ
  3. вң… еЎ«еҶҷй…ҚзҪ®пјҡ
    • рҹҶ” ID: иҮӘе®ҡд№үеҗҚз§°пјҲеҰӮ вҖңntfs-storageвҖқпјү
    • рҹ“Ғ зӣ®еҪ•: еЎ«еҶҷжҢӮиҪҪи·Ҝеҫ„пјҲеҰӮ вҖң/mnt/ntfsвҖқпјү
    • рҹ“Ұ еҶ…е®№: е…ЁйҖүжүҖжңүзұ»еһӢ

  1. вң… зӮ№еҮ»вҖқж·»еҠ вҖқе®ҢжҲҗй…ҚзҪ®

8. й…ҚзҪ® NTFS еҲҶеҢәејҖжңәиҮӘеҠЁжҢӮиҪҪ

8.1 жҹҘзңӢ NTFS еҲҶеҢәзҡ„ UUID жҲ– PARTUUID

8.2 зј–иҫ‘ fstab ж–Үд»¶

ж·»еҠ д»ҘдёӢиЎҢпјҲж №жҚ®е®һйҷ…жғ…еҶөдҝ®ж”№ UUID е’ҢжҢӮиҪҪзӮ№пјүпјҡ

еҸӮж•°иҜҙжҳҺпјҡ

  • рҹ”§ nofail,x-systemd.device-timeout=15s: йҳІжӯўеҗҜеҠЁеӨұиҙҘ
  • рҹӘҹ windows_names: йҷҗеҲ¶ж–Үд»¶еҗҚе…је®№ Windows
  • рҹ‘Ө uid=1000,gid=1000: и®ҫзҪ®ж–Үд»¶жүҖжңүиҖ…пјҲйҖҡеёё1000жҳҜз¬¬дёҖдёӘз”ЁжҲ·зҡ„IDпјү
  • рҹ“қ umask=000: и®ҫзҪ®ж–Үд»¶жқғйҷҗпјҲжүҖжңүз”ЁжҲ·еҸҜиҜ»еҸҜеҶҷеҸҜжү§иЎҢпјү

вҡҷпёҸ дёүгҖҒй«ҳзә§й…ҚзҪ®дёҺдјҳеҢ–

1. жҢӮиҪҪйҖүйЎ№иҜҰи§Ј

еҜ№дәҺ ext4 ж–Үд»¶зі»з»ҹпјҡ

  • вҡЎ noatime: дёҚжӣҙж–°ж–Үд»¶и®ҝй—®ж—¶й—ҙпјҢжҸҗй«ҳжҖ§иғҪ
  • рҹ”„ data=ordered: ж•°жҚ®еҶҷе…ҘйЎәеәҸпјҲе№іиЎЎжҖ§иғҪдёҺе®үе…ЁжҖ§пјү

еҜ№дәҺ NTFS ж–Үд»¶зі»з»ҹпјҡ

  • рҹҡҖ big_writes: еҗҜз”ЁеӨ§еҶҷе…Ҙж“ҚдҪңпјҢжҸҗй«ҳжҖ§иғҪ
  • рҹҮЁрҹҮі locale=zh_CN.UTF-8: и®ҫзҪ®дёӯж–Үlocaleж”ҜжҢҒдёӯж–Үж–Үд»¶еҗҚ

2. жҖ§иғҪдјҳеҢ–е»әи®®

еҜ№дәҺжңәжў°зЎ¬зӣҳпјҡ

еҜ№дәҺ SSD зЎ¬зӣҳпјҡ

  • вҷ»пёҸ discard: еҗҜз”ЁTRIMеҠҹиғҪпјҲд»…йҖӮз”ЁдәҺSSDпјү
  • вҡЎ data=writeback: жҸҗй«ҳжҖ§иғҪпјҲдҪҶеўһеҠ ж•°жҚ®дёўеӨұйЈҺйҷ©пјү

3. еӨҡз”ЁжҲ·жқғйҷҗи®ҫзҪ®

и®ҫзҪ®зү№е®ҡз”ЁжҲ·з»„и®ҝй—®жқғйҷҗпјҡ

жӣҝжҚў 1005 дёә storage-users з»„зҡ„GIDгҖӮ

рҹ”§ еӣӣгҖҒеёёи§Ғй—®йўҳдёҺи§ЈеҶіж–№жЎҲ

1. вқҢ жҢӮиҪҪеӨұиҙҘпјҡжқғйҷҗдёҚи¶і

2. вқҢ жҢӮиҪҪеӨұиҙҘпјҡж–Үд»¶зі»з»ҹжҚҹеқҸ

3. вқҢ дёӯж–Үж–Үд»¶еҗҚжҳҫзӨәд№ұз Ғ

4. вқҢ еҶҷе…ҘйҖҹеәҰж…ў

5. вқҢ ж— жі•еңЁPVE Webз•ҢйқўдёӯзңӢеҲ°еӯҳеӮЁ

рҹ“Ҡ дә”гҖҒж–Үд»¶зі»з»ҹзү№жҖ§еҜ№жҜ”

зү№жҖ§ ext4 NTFS иҜҙжҳҺ
рҹҗ§ Linuxе…је®№жҖ§ вӯҗвӯҗвӯҗвӯҗвӯҗ вӯҗвӯҗвӯҗ ext4жҳҜLinuxеҺҹз”ҹж–Үд»¶зі»з»ҹ
рҹӘҹ Windowsе…је®№жҖ§ вӯҗ (йңҖз¬¬дёүж–№иҪҜд»¶) вӯҗвӯҗвӯҗвӯҗвӯҗ NTFSжҳҜWindowsеҺҹз”ҹж–Үд»¶зі»з»ҹ
рҹ“Ғ еӨ§ж–Үд»¶ж”ҜжҢҒ вӯҗвӯҗвӯҗвӯҗвӯҗ вӯҗвӯҗвӯҗвӯҗвӯҗ дёӨиҖ…йғҪж”ҜжҢҒи¶…еӨ§ж–Үд»¶
рҹ”җ жқғйҷҗз®ЎзҗҶ вӯҗвӯҗвӯҗвӯҗвӯҗ вӯҗвӯҗвӯҗ ext4зҡ„Linuxжқғйҷҗз®ЎзҗҶжӣҙе®Ңе–„
рҹ“қ ж—Ҙеҝ—еҠҹиғҪ вӯҗвӯҗвӯҗвӯҗвӯҗ вӯҗвӯҗвӯҗвӯҗ дёӨиҖ…йғҪжҸҗдҫӣж—Ҙеҝ—еҠҹиғҪ
вҡЎ жҖ§иғҪ вӯҗвӯҗвӯҗвӯҗ вӯҗвӯҗвӯҗ ext4еңЁLinuxдёҠйҖҡеёёжҖ§иғҪжӣҙеҘҪ

рҹ’Ў е…ӯгҖҒжңҖдҪіе®һи·өе»әи®®

  1. рҹ’ҫ еӨҮд»ҪйҮҚиҰҒж•°жҚ®пјҡеңЁж“ҚдҪңеүҚе§Ӣз»ҲеӨҮд»ҪйҮҚиҰҒж•°жҚ®
  2. рҹҶ” дҪҝз”ЁUUIDиҖҢйқһи®ҫеӨҮеҗҚпјҡи®ҫеӨҮеҗҚпјҲеҰӮ/dev/sda1пјүеҸҜиғҪеҸҳеҢ–пјҢUUIDжӣҙзЁіе®ҡ
  3. рҹ§Ә жөӢиҜ•жҢӮиҪҪйҖүйЎ№пјҡеңЁз”ҹдә§зҺҜеўғдҪҝз”ЁеүҚжөӢиҜ•дёҚеҗҢжҢӮиҪҪйҖүйЎ№зҡ„жҖ§иғҪеҪұе“Қ
  4. рҹ”Қ зӣ‘жҺ§зЈҒзӣҳеҒҘеә·пјҡе®ҡжңҹжЈҖжҹҘзЈҒзӣҳSMARTзҠ¶жҖҒе’Ңж–Үд»¶зі»з»ҹеҒҘеә·
  5. рҹ”„ иҖғиҷ‘ж–Үд»¶зі»з»ҹиҪ¬жҚўпјҡеҜ№дәҺй•ҝжңҹеңЁLinuxдҪҝз”Ёзҡ„зЈҒзӣҳпјҢиҖғиҷ‘иҪ¬жҚўдёәext4жҲ–Btrfs

иҪ¬жҚўNTFSеҲ°ext4пјҲж•°жҚ®еӨҮд»ҪеҗҺпјүпјҡ

рҹҡЁ дёғгҖҒжіЁж„ҸдәӢйЎ№дёҺиӯҰе‘Ҡ

  1. вҡЎ Windowsеҝ«йҖҹеҗҜеҠЁпјҡеҰӮжһңNTFSзЎ¬зӣҳжқҘиҮӘWindowsзі»з»ҹпјҢиҜ·зЎ®дҝқзҰҒз”ЁWindowsзҡ„вҖқеҝ«йҖҹеҗҜеҠЁвҖқеҠҹиғҪпјҢеҗҰеҲҷеҸҜиғҪеҜјиҮҙж–Үд»¶зі»з»ҹжҚҹеқҸ
