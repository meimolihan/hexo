Typecho 音乐播放器 QPlayer2

一款简洁小巧的 HTML5 底部悬浮音乐播放器

安装

  1. 下载 或克隆此工程
  2. 将文件夹重命名为 QPlayer2，并放到 Typecho 的插件目录，默认 usr/plugins
  3. 在控制台启用插件

歌曲列表

JSON 数组

键名 说明 必须
name 歌曲名称 是
artist 艺术家数组或文本 否
cover 封面链接 否
lyrics [歌词文本][0] 否
provider 提供器名称 否，默认 [default][1]
lrc 歌词链接，随提供器而定 否
