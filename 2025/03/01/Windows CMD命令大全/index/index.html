Windows CMD命令大全

前言

Windows命令提示符（CMD）是许多Windows用户常用的工具，能够通过命令行方式与操作系统直接交互。为你整理了一份必备的Windows CMD命令大全，以备不时之需。

一、文件和目录管理命令

命令 描述 示例
dir 显示当前目录下的文件和文件夹列表 dir
cd 切换当前目录 cd C:\Users\
mkdir 创建新文件夹 mkdir new_folder
rmdir 删除空目录 rmdir folder_name
del 删除文件 del filename.txt
copy 复制文件或文件夹 copy source.txt destination.txt
move 移动文件或文件夹 move file.txt D:\NewFolder\
rename 重命名文件或文件夹 rename oldname.txt newname.txt

二、网络命令

命令 描述 示例
ping 测试网络连接是否通畅 ping www.baidu.com
ipconfig 显示网络配置和IP地址 ipconfig
tracert 路由追踪，显示数据包经过的路径 tracert www.baidu.com
netstat 显示当前的网络连接状态 netstat -a
nslookup 查询DNS记录 nslookup www.baidu.com

三、系统信息和管理命令

命令 描述 示例
systeminfo 查看系统的详细信息，包括操作系统版本、内存等 systeminfo
tasklist 显示当前运行的所有进程 tasklist
taskkill 结束指定的进程 taskkill /f /im notepad.exe
chkdsk 检查并修复磁盘错误 chkdsk C:
sfc 系统文件检查，修复损坏的系统文件 sfc /scannow

四、磁盘管理命令

命令 描述 示例
diskpart 打开磁盘分区管理工具 diskpart
format 格式化磁盘 format D: /fs:ntfs
vol 显示磁盘卷标 vol C:
wmic logicaldisk get name 获取所有磁盘的盘符 wmic logicaldisk get name

五、文件查找与内容搜索命令

命令 描述 示例
find 查找文件中包含特定字符串的行 find “hello” filename.txt
findstr 高级文件内容搜索，支持正则表达式 findstr /s “hello” *.txt
dir /s /b 查找指定文件 dir C:\ /s /b \ find “filename.txt”

六、系统关机与重启命令

命令 描述 示例
shutdown 关机或重启计算机 shutdown /s /t 0 # 立即关机
shutdown /r /t 0 # 立即重启
exit 退出当前命令提示符窗口 exit

七、批处理文件和脚本命令

命令 描述 示例
echo 打印文本或启用/禁用命令回显 echo Hello, World!
pause 在批处理文件中暂停，等待用户按键继续 pause
exit 退出批处理文件 exit
call 调用其他批处理文件 call other_batch.bat

八、其他实用命令

命令 描述 示例
cls 清除屏幕内容 cls
time 显示或设置系统时间 time
date 显示或设置系统日期 date
exit 退出当前命令提示符窗口 exit
