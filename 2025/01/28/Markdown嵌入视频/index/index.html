第一部分 、视频链接参数部分

  1. 原视频链接参数部分：

  1. 添加的内容说明如下：
  • high_quality 1 最高画质 0 最低画质
  • danmaku 1 打开弹幕 0 关闭弹幕
  • autoplay 1 打开自动播放 0 关闭自动播放

这是《视频链接参数》修改后的样子

<iframe src="//player.bilibili.com/player.html?isOutside=true&aid=327623069&bvid=BV1JA411h7Gw&cid=171385214&p=1&high_quality=1&danmaku=0&autoplay=0"

第二部分 、iframe 参数部分

  1. 原视频的 “iframe” 参数:
    scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true"></iframe>
  2. 将上面的iframe 参数，在 "true" 后面 加空格粘贴以下内容：
  1. 添加的内容说明如下：
  • allowfullscreen 允许全屏 **true **允许 **false **不允许
  • width 宽度像素
  • height 高度像素
  • sandbox=”allow-top-navigation allow-same-origin allow-forms allow-scripts “禁止弹出网页
